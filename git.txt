git是一个分布式的版本管理软件，不同于SVN，它是分布式的，也就是说每台电脑都保存有各个版本的代码，一个用户提交代码时，会将提交的代码放到本地一份和中心一份

git的安装

git使用的步骤：
	1、进入要初始化的目录
	2、git init :初始化，即让git帮助我们管理当前文件夹
	3、git status：检测当前目录下文件的状态
	4、三种状态的变化:
		红色：新增的文件/修改了原文件  -> git add filename
 		绿色：git已经管理起来的文件  -> git  commit -m "描述信息"
		生成版本
		

git的一些命令及注意事项：
	1、查看版本命令：git --version
	2、初始化：git init
	3、查看文件夹中文件中状态：git status  (红色：没有git被管理，绿色：已经被git管理）
               可以执行  git add filename指令将没有被管理的文件进行管理，git add .指令可以将当前文件夹下没有被git管理的文件全部进行管理
	4、当我们修改文件的内容后，再执行git status指令时，被修改的内容会变红，我们需要再次执行git add .指令，然后执行git commit -m "xxx"
	5、查看版本记录：git log
	6、回滚：git reset --hard  版本号，如果进行回滚后，则之前的的版本则看不到了，但如果又想回到之前的版本则可以使用git reflog命令查看记录，找到版本号则可以恢复到之前的版本
